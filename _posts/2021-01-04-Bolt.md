# Scanning
## nmap
We'll start with scanning the host for open ports using [nmap](https://nmap.org/). The command we'll use is `sudo nmap -sV -T4 -p- -O -oN nmap 10.10.255.222`. Let's break it down:
* `-sV` determine service/version info
* `-T4` for faster execution
* `-p-` scan all ports
* `-O` identify Operating System
* `-oN` output to file, in our case it's called nmap

```console
$ sudo nmap -sV -T4 -p- -O -oN nmap 10.10.255.222
Starting Nmap 7.91 ( https://nmap.org ) at 2020-12-31 10:38 EST
```
